Oh My Pi Titanium

A consolidated terminal theme repo with one Titanium palette, portable config outputs, and a live Windows Terminal installer.

Install GitHub

Install

Use the native installer where the terminal supports it, or generate portable artifacts for checked-in dotfiles.

iTerm2

python3 terminal-theme.py --set-default
Writes an iTerm2 Dynamic Profile and can set it as the default profile.

Windows

pwsh -ExecutionPolicy Bypass -File .\make-profile.ps1 -InstallOhMyPosh -InstallFont
Patches settings.json, backs it up, installs optional prompt and font support.

Artifacts

python3 terminal-theme.py --target all --output-dir dist
Builds iTerm2, WezTerm, Windows Terminal, Orca, and PowerShell files together.

Outputs

The generator writes terminal-native files instead of forcing one format into every app.

JSON

iTerm2 Dynamic Profile

Foreground, background, cursor, selection, ANSI colors, font, spacing, and optional backdrop fields.

Lua

WezTerm

A named color scheme with Titanium ANSI arrays and JetBrains Mono Nerd Font settings.

JSON

Windows Terminal

A settings fragment for portable config and a PowerShell installer for live settings mutation.

JSON

Orca

A settings fragment that maps Titanium colors to Orca terminal color overrides.

PS1

PowerShell

Profile styling for host messages while the terminal owns the full ANSI palette.
